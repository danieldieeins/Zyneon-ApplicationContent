News

Kommende Features

• Instanzpfad-Verschiebung, damit man die Möglichkeit hat, sich auszusuchen, wo die Instanzen installiert sind.
• Englische Version der Application
• Automatische Spracherkennung des Betriebssystems.
• Instanzen-Ersteller, damit man eigene Vanilla, Fabric und Forge Instanzen erstellen kann.
• Instanzen-Importer, damit man von anderen erstellte Instanzen per ID importieren kann.
• Zyneon+ 1.20.2 (kommt mit Public Beta 7)
• Entfernung von Zyneon+ 1.20.1 (kommt mit Public Beta 7)

• und potenziell noch vieles mehr. 

Public Beta 5

25.09.2023: Die Application wurde auf die Public Beta 5 aktualisiert!
Diese Version hat einige unter-der-Haube Veränderungen, Bugfixes und folgende Features

Hinzugefügt:
- Automatisches Aktualisieren beim An- und Abmelden.
- Standard RAM-Einstellungen speichern (Bugfix)
- UI Update 122: Start/News aktualisiert

Public Beta 4

24.09.2023: Die Application wurde auf die Public Beta 4 aktualisiert!
Diese Version bringt eine Reihe von Backend Performance Updates und einige Bugfixes mit sich.

Hinzugefügt:
- Start/News Seite im Launcher mit diesen Informationen.
- RAM-Einstellungen (Bugfix)
- Zyneon Application Logo in der Taskleiste (Bugfix)
- Zyneon Application Logo in der Titelleiste (Bugfix)
- 1.16.5-Button für Zyneon+
- Vorbereitung auf die Instanzpfad-Verschiebung

Entfernt:
- Instanzen-Dropdown-Menü (Bug)

Public Beta 3

23.09.2023: Performance Update im Backend, keine direkten Neuerungen.

Public Beta 2

23.09.2023: Die Installer wurden aktualisiert!
Die Application wurde nun auf Java 21 aktualisiert, zudem wurden ältere Java Versionen hinzugefügt, um mehr Kompatibilität zu bieten.

Hinzugefügt:
- Java 21: für die Application und Minecraft Versionen über 1.16.5
- Java 11: für Minecraft Versionen 1.13, 1.13.1, 1.13.2, 1.14, 1.14.1, 1.14.2, 1.14.3, 1.14.4, 1.15, 1.15.1, 1.15.2, 1.16, 1.16.1, 1.16.2, 1.16.3, 1.16.4 und 1.16.5
- Java 8: für Minecraft Versionen unter 1.13

Entfernt:
- RAM-Einstellungen (Bug)
